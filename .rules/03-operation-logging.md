# 03 - 自动操作日志记录 (Operation Logging)

每当 AI 助手完成一次实质性的操作（包括但不限于代码修改、文件创建、命令运行、Git 提交等），**必须**立即更新项目根目录下的 `walkthrough.md` 文件。

### 强制要求
1. **即时性**：操作完成后，在告知用户之前，优先更新日志。
2. **准确性**：必须真实记录所做的每一步改动，包括产生错误的尝试及其修正过程。
3. **关联性**：如果之前的任务有更新，应在日志中注明关联的旧条目。
4. **归档位置**：目标文件始终为 `d:\8-python-project\Cinemind\walkthrough.md`。

> [!IMPORTANT]
> 记录日志不是可选的，而是工作流程中不可分割的一部分。它旨在让用户随时掌握项目的历史变迁。
